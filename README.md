## Getting Started

### Prerequisites

- [Bun](https://bun.sh/) installed on your machine

### Installation

```bash
# Clone the repository
git clone https://github.com/S1933/bun-pwa.git
cd bun-pwa

# Install dependencies
bun install
```

### Development

```bash
bun run dev
```

### Build

```bash
bun run build
```

### Preview Production Build

```bash
bun run preview
```

## Tech Stack

- **Runtime**: Bun
- **Build Tool**: Vite
- **Framework**: React 18
- **PWA**: vite-plugin-pwa
